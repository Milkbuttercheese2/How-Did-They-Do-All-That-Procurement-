# 검사 이의와 재검사 요청 — 조사 워크시트 (2026-07-24)

slug: reinspection-request / type: 이행·대금 / priority: 67 / 주 이용자: 조달업체(계약상대자)

기존 제도 `inspection-and-acceptance`(검사와 검수)의 하위·인접 절차. 겹치지 않도록 **검사 결과에 대한 이의 → 재검사 → 시정 → 검사기간 재기산 → 지체상금 산입**의 불복·구제 미시절차에 초점을 두었다.

## 법령 간 관계
- 법 제14조(검사)·제15조(대가 지급)가 상위 근거이고, 시행령 제55조가 검사기간(14일)·시정 시 재기산을 정한다.
- 검사 이의·재검사의 구체 절차는 각 계약예규(물품·공사·용역 일반조건)에 있어 규칙 8·12(조달청/기재부 예규 우선)에 따라 예규 조문을 제1근거로 인용하고 법률·시행령을 근거 사슬로 병기.
- 재검사·시정으로 이행기간이 연장되면 각 예규의 지체상금 조문(물품 제24조, 공사 제25조, 용역 제18조)의 지체일수 산정 규정으로 연결된다.

## 노드 최적 근거·돌려쓰기 점검(규칙 21)
- 물품 제19조제7항: P02·P03 2개 노드(≤2). 물품 제19조제4항: P05·P06(≤2). 물품 제19조제8항: P01·P07(≤2). 공사 제27조제6항: P01·P02(≤2). 모두 조 단위 3노드 미만.
- P04(재검사 실시) 제1근거는 시행령 제55조제1항(검사기간)로 두어 제19조제7항 3노드 재사용을 회피.
