## 조사 방식 메모
- 근거 법령 원문 캐시(`sources/law-cache/조달사업에-관한-법률(.md/-시행령.md/-시행규칙.md)`) 중 **법률·시행령은 목차(조문 번호·제목)만** 담겨 있고 본문은 없었다. **시행규칙만 조문 본문(제12조·제13조 포함)이 캐시에 있어 항 단위까지 직접 대조**했다.
- korean-law MCP `get_law_text`/`legal_research(law_system)`는 이번 세션 내내 조달사업법(mst 277155)에 대해 `[NOT_FOUND]`/`[EXTERNAL_API_ERROR] LspttnThdCmpLawXService`를 반환해 법률·시행령 본문을 직접 대조하지 못했다. `search_law`는 정상 동작(현행 mst 확인).
- 따라서 **조문 번호·제목은 캐시 목차(국가법령정보센터 Open API 사본)로 확정**, **법률·시행령의 내용 취지는 WebSearch 교차 확인**, **시행규칙 제12·13조는 캐시 원문 대조**로 구분해 기록한다. 캐시 목차·본문으로 확정 못 한 항(제N항)은 회피하고 조(제N조) 단위로만 인용한다.
- 조문 실존 기계 대조와 항 단위 확정은 통합 시 관리자 재검증에 맡긴다. verification.status = needs-review.
- 시행예정 개정 주의: 현행 시행일 2026-01-02 기준. 조달사업법에 2026-09-11 시행(제21420호)·2026-09-03 시행(제21715호) 개정 예정본이 있어 조번호·내용 변동 가능성 있음 → verification.notes에 기재.

## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것
- 법률·시행령의 **항(제N항) 번호**는 본문 미대조 → 법 제29·30·32조는 취지 서술 시 항 표기를 process legal_basis에서 조 단위 중심으로 쓰고, 확신 낮은 항 구분은 fieldVerification로.
- 시행령 제36조(등록제한 기간·요건), 제39조(민관 공동 비축 참여대상), 제40조(민관 공동 비축협약), 제41조(비축시설 사용료 감면)는 캐시 목차만 확인 → canvas legalBasis 조 단위 인용에 그치고 process 노드 근거로는 uncheckable 처리.
- **비축대상 품목 목록**(알루미늄·구리·아연·니켈·희소금속 등)은 재정경제부장관(기획재정부) 고시 값으로 WebSearch(pps)만 근거 → 본문에서는 "재정경제부장관 고시로 지정" 수준만 쓰고 구체 품목은 fieldVerification.
- **주간 판매계획·업체별 한도량, 실수요 확인 서류(완제품성분분석표 등), 방출 신청 창구(지방청/나라장터)**는 조달청 비축사업 운영규정·pps 안내 기반 실무 → 조번호 없이 절차/fieldVerification로만 기술.
- 판매가격 구매원가 초과분의 조달특별회계 세입 처리는 WebSearch(제30조 관련)만 근거 → 요지에 넣지 않고 워크시트에만 기록.
